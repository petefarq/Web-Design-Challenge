Summary: Lattitude vs. Weather

The purpose of this project was to analyze how weather relates to lattitude. To accomplish this we first pulled data from the OpenWeatherMap API to assemble a dataset of over 500 cities.

We then used Matplotlib to plot various attributes of the weather and see how they related to lattitude. Factors we examined included temperature, wind speed, humidity, and cloudiness. This site provides the source data and visualizations as well as analysis of any trends and correlations.
